    Welcome to Gir.Core #

    Gir.Core is a project which aims to provide C# bindings for different GObject based libraries like GTK for user interfaces.

    The focus is to provide a C# friendly API and a corresponding online documentation.

    What do you want to do next?

    • If you want to get more details before starting to program take a look at the apps section or the supported libraries .
    • If you want to get to the nuget packages see our nuget organization for an overview of all available packages.
    • If you want to contribute to the project see the github project and checkout the contribution guidelines .
